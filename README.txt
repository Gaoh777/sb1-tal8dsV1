个人爱收藏网站使用说明

1. 功能概述
   - 添加和管理书签
   - 分类显示书签
   - 常用网址快速访问
   - 搜索书签
   - 高级搜索
   - 导入/导出书签
   - 主题设置
   - 深色模式切换
   - 删除书签管理
   - 书签排序功能
   - 标签管理
   - 书签统计信息

2. 如何操作

   2.1 添加新书签
   - 点击顶部的"来收藏吧"按钮
   - 在弹出的表单中填写网站名称、URL和选择标签
   - 如需添加新标签，在"新增标签"输入框中输入并点击"添加"
   - 点击"保存收藏"按钮完成添加

   2.2 查看和使用书签
   - 书签按标签分类显示在主页面
   - 点击书签可直接访问对应网站
   - 常用网址会显示在页面顶部，方便快速访问

   2.3 搜索书签
   - 在顶部搜索框输入关键词
   - 搜索结果会实时显示在下方

   2.4 高级搜索
   - 点击搜索框右侧的"高级搜索"按钮
   - 在弹出的面板中设置搜索条件，包括名称、网址、标签、创建日期范围
   - 选择排序方式和顺序
   - 点击"搜索"按钮查看结果

   2.5 编辑书签
   - 将鼠标悬停在书签上，点击编辑图标
   - 修改书签信息后保存

   2.6 删除书签
   - 将鼠标悬停在书签上，点击删除图标
   - 删除的书签可在"已删除的收藏"中查看和恢复

   2.7 导出书签
   - 点击顶部的设置图标，选择"导出收藏"
   - 选择保存位置，即可导出为Excel文件

   2.8 导入书签
   - 点击顶部的设置图标，选择"导入收藏"
   - 选择之前导出的Excel文件或按模板格式创建的文件

   2.9 主题设置
   - 点击顶部的设置图标，选择"设置主题颜色"
   - 在弹出的面板中自定义背景颜色、图片等

   2.10 切换深色模式
   - 点击右上角的太阳/月亮图标即可切换

   2.11 管理已删除的书签
   - 点击顶部的设置图标，选择"查看已删除的收藏"
   - 在弹出的面板中可以恢复或永久删除书签

   2.12 管理标签
   - 点击顶部的设置图标，选择"管理标签"
   - 在弹出的面板中，您可以：
     - 编辑标签名称：点击标签旁的编辑图标，输入新名称后保存
     - 删除标签：点击标签旁的删除图标
     - 合并标签：选择源标签和目标标签，然后点击"合并"按钮

   2.13 查看统计信息
   - 点击顶部的设置图标，选择"查看统计信息"
   - 在弹出的面板中查看总体统计、最近添加的书签和标签使用情况

   2.14 从常用网址中移除
   - 将鼠标悬停在常用网址上，点击右上角的删除图标
   - 这只会将书签从常用网址中移除，不会删除书签本身

3. 注意事项
   - 删除书签不会立即永久删除，可在"已删除的收藏"中恢复
   - 导入书签时，请确保Excel文件格式与导出格式一致
   - 主题设置中上传的图片会保存在本地存储中，刷新页面不会丢失
   - 深色模式设置会被保存，下次打开网站时会保持之前的设置
   - 删除标签会同时删除使用该标签的所有书签，请谨慎操作
   - 合并标签后，原标签下的所有书签将移动到新标签下
   - 从常用网址中移除书签不会影响该书签在其他分类中的显示
